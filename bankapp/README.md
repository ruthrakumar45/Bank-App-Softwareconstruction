# Bank Management System

A full-stack Bank Management System built using Java, Spring Boot, MySQL, HTML, CSS, and JavaScript.  
This project helps manage customer accounts, transactions, and banking operations through a clean web interface.

---

## Features

- User Registration & Login
- Create Bank Accounts
- Deposit Money
- Withdraw Money
- Balance Checking
- Transaction History
- Secure Backend using Spring Boot
- MySQL Database Integration
- REST API Architecture

---
